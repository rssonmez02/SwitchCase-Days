# SwitchCase-Days
# 📅 C# ile Haftalık Takvim Yapısı (Day of the Week Evaluator)

Bu proje, C# programlama dilinde mantıksal sorguların (`if-else` / `switch-case`) kullanımını, kullanıcıdan veri alma süreçlerini ve veri tipi dönüşümlerini (Type Casting) pekiştirmek amacıyla geliştirilmiş bir konsol uygulamasıdır.

## 🚀 Özellikler
* **Kullanıcı Etkileşimi:** Kullanıcıdan 1-7 arasında sayısal girdi alır.
* **Koşullu Algoritma Yapısı:** Girilen sayısal değeri mantıksal sorgulardan geçirerek doğru haftanın gününe eşler.
* **Hata Yönetimi (Geliştirilebilir):** Tanımlanan aralık dışındaki girdiler için geçersiz işlem bildirimi yapar.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** C# (.NET)
* **Geliştirme Ortamı:** Visual Studio 2013

## 💻 Nasıl Çalıştırılır?
1. Bu depoyu (repository) bilgisayarınıza indirin veya klonlayın.
2. Projeyi Visual Studio ile açın.
3. `F5` veya `Ctrl + F5` tuş kombinasyonunu kullanarak uygulamayı çalıştırın.
